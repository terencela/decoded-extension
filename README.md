# Decoded — LinkedIn Translator

A Chrome Extension (Manifest V3) that translates corporate speak and AI-generated content on LinkedIn in real time.

## Features
- 5 behavioral archetypes classifier
- AI saturation score (0-100%)
- Shareable 1200×630 decode cards
- AI-generated comment detection
- Free: 5 decodes/day | Pro: $3/mo | Power: $5/mo
